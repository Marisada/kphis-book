# KPHIS-RS book
โปรแกรม [KPHIS](https://gitlab.kph.go.th/apichat/kphisdockerimages.git) เป็นโปรแกรม IPD Paperless แบบ Open Source ที่พัฒนาโดย รพ.กำแพงเพชร ([Facebook](https://www.facebook.com/KPHIS-101091131902046)) และมีการใช้งานอย่างแพร่หลายในประเทศไทย

ในปี พ.ศ.2566 รพ.นาดูน จ.มหาสารคาม ได้นำ KPHIS version 24.01.2 มาพัฒนาด้วยภาษา [Rust](https://www.rust-lang.org/) เป็น KPHIS-RS
* `version 0.1.x` : ใช้ User Interface และโครงสร้างฐานข้อมูลเดิม
* `version 0.2.x` : ปรับปรุงฐานข้อมูล เพิ่มระบบการใช้งานใหม่, ปรับปรุง User Interface
* `version 0.3.x` : ปรับปรุงฐานข้อมูล ให้สามารถ Admit ใน KPHIS ก่อนใน HOSxP ได้
* `version 0.4.x` : ปรับปรุงฐานข้อมูล ให้สามารถสร้าง Custom report ได้

ทางทีมงานได้สร้างคู่มือในการใช้งานโปรแกรม KPHIS-RS นี้ขี้น เพื่ออำนวยความสะดวกในการศึกษาวิธีการใช้งานโดยทั่วไป

เยี่ยมชมคู่มือฉบับนี้ได้ที่ [GitHub Page](https://marisada.github.io/kphis-book/)
